# Positive? - Function Summary

## Summary:

Returns TRUE if the value is positive.

## Usage:

**positive? number**

## Arguments:

**number** - The number argument. (must be: number char money time)

## Description:

Returns FALSE for all other values.

```
print positive? 50
true
```

```
print positive? -50
false
```
