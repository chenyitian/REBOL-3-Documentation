# ** - Function Summary

## Summary:

Returns the first number raised to the second number.

## Usage:

***\* number exponent**

## Arguments:

**number** - The number argument. (must be: number)

**exponent** - The exponent argument. (must be: number)

## Description:

This is the operator for the POWER function.

```
print 10 ** 2
100.0
```

```
print 12.345 ** 5
286718.338524636
```
