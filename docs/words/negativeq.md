# Negative? - Function Summary

## Summary:

Returns TRUE if the number is negative.

## Usage:

**negative? number**

## Arguments:

**number** - The number argument. (must be: number char money time)

## Description:

Returns FALSE for all other values.

```
print negative? -50
true
```

```
print negative? 50
false
```
