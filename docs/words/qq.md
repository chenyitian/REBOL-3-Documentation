# ?? - Function Summary

## Summary:

Prints a variable name followed by its molded value. (for debugging)

## Usage:

**?? name**

## Arguments:

**name** - The name argument.

## Description:

Handy for printing the value of a variable while debugging.

```
test: [123 "testing"]
?? test
test: [123 "testing"]
```
