# Zero? - Function Summary

## Summary:

Returns TRUE if the number is zero.

## Usage:

**zero? number**

## Arguments:

**number** - The number argument. (must be: number pair char money time tuple)

## Description:

Check the value of a word is zero.

```
print zero? 50
false
```

```
print zero? 0
true
```
